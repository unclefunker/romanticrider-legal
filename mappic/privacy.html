맵픽(mappic) 개인정보처리방침

시행일: 2026년 7월 7일 · 버전 1.0

낭만라이더(이하 "운영자")는 맵픽(mappic) 앱(이하 "서비스") 이용자의 개인정보를 소중히 여기며, 「개인정보 보호법」, 「위치정보의 보호 및 이용 등에 관한 법률」 등 관련 법령을 준수합니다. 본 방침은 서비스가 어떤 정보를 어떻게 처리하는지 설명합니다.

한눈에 보기

  • 맵픽은 회원가입·로그인이 없으며, 운영자 서버로 개인정보를 수집·전송하지 않습니다.
  • 사진, 위치, 메모 등 모든 데이터는 이용자의 기기 안에만 저장됩니다.
  • 주소 표시·근처 명소 조회를 위해 좌표가 외부 API에 일시적으로 전송되지만, 운영자는 이를 저장하지 않습니다.
  • 광고 SDK·분석(트래킹) 도구를 사용하지 않습니다.

1. 처리하는 정보와 목적

서비스는 아래 정보를 이용자 기기 안에서만 처리하며, 운영자 서버로 수집하지 않습니다.

정보이용 목적저장 위치·기간
카메라 영상(촬영 이미지) 사진 촬영 및 타임스탬프·위치 오버레이 합성 기기 내 저장 · 이용자가 삭제할 때까지
위치정보(GPS 좌표) 촬영 시점 좌표 기록, 사진의 지도 표시, 주소 변환, 근처 명소 조회 기기 내 저장 · 이용자가 삭제할 때까지
사진 라이브러리(갤러리) 접근 위치정보가 포함된 기존 사진을 지도 다이어리로 불러오기(이용자가 실행한 경우) 기기 내 저장 · 이용자가 삭제할 때까지

각 권한(카메라·위치·사진)은 해당 기능을 처음 사용할 때 이용자 동의를 받아 활성화되며, 기기 설정에서 언제든 철회할 수 있습니다. 권한을 거부해도 해당 기능 외의 서비스 이용은 가능합니다.

2. 외부 API로의 좌표 전송

아래 기능을 실행할 때, 기능 제공에 필요한 좌표가 이용자 기기에서 외부 API로 직접 전송됩니다. 전송된 좌표는 해당 기능의 응답 생성에만 사용되며, 운영자는 이를 수집·저장하지 않습니다.

수신자기능전송 항목
네이버클라우드(주) — 네이버 지도 API 지도 화면 표시, 좌표→주소 변환(역지오코딩) 지도 뷰포트·사진 촬영 좌표
한국관광공사 — TourAPI 현재 위치 주변 관광 명소 조회 현재 위치 좌표

각 수신자의 개인정보 처리에 관한 사항은 해당 기관의 개인정보처리방침을 따릅니다.

3. 제3자 제공·처리위탁·국외이전

운영자는 이용자의 개인정보를 제3자에게 제공하거나, 처리를 위탁하거나, 국외로 이전하지 않습니다.

4. 보유 기간 및 파기

모든 데이터는 이용자 기기에만 저장되므로, 이용자가 앱 안에서 사진·기록을 삭제하거나 앱을 삭제하면 즉시 파기됩니다. 운영자는 별도로 보유하는 개인정보가 없습니다.

5. 이용자의 권리

  • 기기 설정에서 카메라·위치·사진 권한을 언제든 허용·철회할 수 있습니다.
  • 앱 안에서 개별 사진과 기록을 직접 삭제할 수 있습니다.
  • 앱 삭제 시 앱이 저장한 모든 데이터가 함께 삭제됩니다.

6. 안전성 확보 조치

  • 데이터를 운영자 서버에 두지 않는 온디바이스 구조로 설계했습니다.
  • 외부 API 통신은 HTTPS 암호화 구간으로만 이루어집니다.

7. 아동의 개인정보

서비스는 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 광고·분석 도구

서비스는 현재 광고 SDK 및 행태 분석(트래킹) 도구를 사용하지 않습니다. 향후 도입하는 경우 본 방침을 개정하고 앱 내 공지를 통해 사전 안내합니다.
